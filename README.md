# 🚀 PlacementPro

**PlacementPro** is a modern, enterprise-grade Student & Career Management System. It streamlines the recruitment lifecycle by bridging the gap between academic institutions and corporate hiring partners. Built with a focus on scalability and clean architecture, it provides a centralized hub for managing student profiles, job applications, and placement analytics.

---

## 🛠️ Tech Stack

* **Language:** Java 17 (LTS)
* **Framework:** Spring Boot 3.3.5
* **Persistence:** Spring Data JPA
* **Database:** PostgreSQL
* **Utilities:** Project Lombok (Clean Code)
* **Build Tool:** Maven

---

## ✨ Core Features

* **Student Lifecycle Management:** Detailed profile tracking including academic history and skills.
* **Placement Tracking:** Real-time monitoring of job postings, applications, and interview statuses.
* **RESTful API:** Clean, decoupled endpoints ready for MERN or mobile frontend integration.
